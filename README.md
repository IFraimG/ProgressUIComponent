
### Разработка прототипа блока Progress для использования в мобильных web-приложениях.

Основное предназначение блока - отображение процесса выполнения процессов и их прогресса выполнения.

____

### Структура проекта

1. index.html - демо страница с версткой компонентов для тестирования Progress и его инициализация
2. /demo - содержатся файлы index.js и style.css, которые являются реализацией тестирования компонента. Функционал прототипа Progress не зависит от них.
3. /lib - содержатся файлы реализации прототипа Progress
